# ROM Exchange CLI

A simple CLI to fetch item prices from the [ROM Exchange](https://www.romexchange.com/) API.

## Installation

```bash
$ wget https://github.com/Log1x/rom-cli/releases/download/v1.0.0/rom-cli
$ chmod +x rom-cli
$ mv rom-cli /usr/local/bin/rom
```

## Usage

```bash
$ rom search <item>
```
